# AliuHook

Java Xposed Api for [LSPlant](https://github.com/LSPosed/LSPlant)

# Supported Android versions
- Android 5.0 - 13 (API level 21 - 33)
- armeabi-v7a, arm64-v8a, x86, x86-64
